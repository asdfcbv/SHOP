# SHOP

## Описание
Проект позиционируется как пример оформления онлайн магазина.

## Как работает и выглядит?
Для проекта были использованы: Python(django), HTML, CSS
1.Главная страница магазина с карточками товаров.
2.Категории с товарами.
3.Описание товара.
4.Корзина с товаром.

Назавание файла | Содержание файла
----------------|-----------------
views.py        | Главный файл
urls.py         | Связывающий файл
templates       | Папка с HTML файлами
settings.py     | Настройки
